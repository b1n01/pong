# Pong

A pong-like written in Lua with [LÖVE](https://love2d.org/) in less than 300 lines.

## Run

Download LÖVE from [https://love2d.org/#download](https://love2d.org/#download) and run:

```bash
git clone https://github.com/b1n01/pong.git
love pong
```
